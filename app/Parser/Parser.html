Get Timetable Data Automatically

Caution: This feature is almost certainly broken in many serious ways. If you are using this to make real decisions please make sure you check the data that this tool generates before using it.
How to use this tool:
  1. Go to UON's timetabling search tool
  2. Select ONE course to view and click View Timetable
  3. In the timetable grid view (not the visual schedule view) right click and click View Page Source
  4. Select all (CTRL + a) and copy (CTRL + c)
  5. Paste into the text field below and submit.
  6. Be sure to check that the data is correct before using the timetable.
